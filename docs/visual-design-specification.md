# Визуальная спецификация и пользовательский опыт «Добрые дела Росатома»

## Общая концепция дизайна

### Философия дизайна
- **Минимализм с корпоративным характером**: чистый, современный дизайн с акцентом на контент
- **Доступность**: интуитивный интерфейс для пользователей всех возрастов
- **Доверие**: корпоративный стиль Росатома, вызывающий уверенность
- **Эмоциональная связь**: теплые цвета и дружелюбная подача

### Визуальная иерархия
1. **Первый уровень**: Заголовки H1, главные CTA кнопки, логотип
2. **Второй уровень**: Заголовки H2, карточки НКО, новости
3. **Третий уровень**: Описания, метаданные, вторичные кнопки
4. **Четвертый уровень**: Подписи, вспомогательный текст

## Адаптивный дизайн

### Мобильные устройства (320-767px)
- Одноколоночная сетка
- Уменьшенные отступы (16px вместо 24px)
- Крупные кнопки (min-height 48px)
- Упрощенная навигация (бургер-меню)
- Уменьшенные шрифты (H1: 28px, H2: 20px)

### Планшеты (768-1023px)
- Двухколоночная сетка для карточек
- Средние отступы (20px)
- Горизонтальная навигация
- Средние шрифты (H1: 32px, H2: 24px)

### Десктоп (1024px+)
- Трехколоночная сетка для карточек
- Полные отступы (24px)
- Полноценная навигация
- Полные шрифты (H1: 38px, H2: 28px)

## Микровзаимодействия

### Hover эффекты
- Кнопки: подъем на 2px, усиление тени
- Карточки: подъем на 4px, усиление тени
- Ссылки: подчеркивание, изменение цвета
- Иконки: легкое масштабирование (1.1x)

### Loading состояния
- Скелетоны для карточек
- Спиннеры для кнопок
- Прогресс-бары для загрузки контента

### Фидбек пользователю
- Успешные действия: зеленая подсветка
- Ошибки: красная подсветка, встряхивание
- Предупреждения: желтая подсветка
